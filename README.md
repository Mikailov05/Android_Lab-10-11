# Student Material Design

Приложение для отображения списка студентов с использованием принципов Material Design 3 и Jetpack Compose.

## Цель работы

Изучение принципов Material Design в Android-приложениях с использованием Jetpack Compose. В ходе работы было создано приложение «Список студентов», демонстрирующее использование цветовых схем, типографики, форм и анимаций Material Design.

## Функциональность

-  Отображение списка студентов с именами и возрастом
-  Круглые аватарки студентов с правильным кадрированием изображений
-  Раскрывающееся описание для каждого студента с анимацией
-  Поддержка светлой и тёмной темы
-  Кастомная цветовая схема (зелёная)
- Пользовательские шрифты (Abril Fatface и Montserrat)
-  Верхняя панель приложения с логотипом и названием
-  Material Design карточки с диагональным скруглением углов
-  Адаптивный интерфейс для разных размеров экрана

## Технологии

- **Язык программирования:** Kotlin
- **UI Toolkit:** Jetpack Compose
- **Дизайн-система:** Material Design 3
- **Архитектура:** MVI (Model-View-Intent)
- **Библиотеки:**
    - androidx.compose.material3 — компоненты Material Design 3
    - androidx.compose.material.icons — иконки Material
    - androidx.compose.animation — анимации
    - androidx.compose.ui.tooling — инструменты предпросмотра
